# 디자인 토큰

정의 원본은 **`src/app/index.css` 의 `@theme`**. 아래 값들은 실제 CSS 변수에서 읽어와 렌더한 거라 **코드와 항상 일치**한다.

공통: 폰트 **Pretendard Variable** · 자간 **-2%** · 행간 **140%**. 간격은 Tailwind 기본(1 = 4px).
